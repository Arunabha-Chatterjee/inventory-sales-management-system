# **Web-Based Inventory and Sales Management System	**



This is a web-based application developed to streamline and manage business operations through a centralized platform. It provides an interactive dashboard for viewing business data over different time periods and monitoring recent activities. The system enables efficient management of business records by allowing users to view and manage product, customer, and invoice information along with their related details and summaries. It also supports maintaining and updating customer records and tracking transactions associated with products and invoices.	                                  



## **Features**

-   **Interactive dashboard** for visualizing **business data**, **key metrics**, and **recent activities**.  
-   Efficient management of **products**, **customers**, and **invoices** with **transaction tracking** and **summaries**.
-   **Secure authentication** and **role-based access control** using **JWT** and **Spring Security**.
-   **Modern, responsive UI** built with **React** and **Tailwind CSS**, supporting **form handling** and **validation** via **React Hook Form**.
-   **Seamless API integration** between **frontend** and **backend** using **Axios**, with **real-time data updates**.
-   **Centralized and scalable platform** for managing all **core business operations** and supporting future **enhancements**.



## **Tools and Technologies**

### **Frontend**
-   **React** – For building dynamic user interfaces.
-   **Tailwind CSS** – For responsive and modern styling.
-   **Axios** – For seamless API integration between frontend and backend.

### **Backend**
-   **Spring Boot** – Framework for building RESTful APIs and backend logic.
-   **Spring Security** – For authentication and access control.
-   **JWT (JSON Web Tokens)** – For secure user authentication.

### **Database**
- **MongoDB** – Database for storing and managing business data.

### **Tools**
-   **IntelliJ IDEA** – Backend development IDE.
-   **VS Code** – Frontend development editor.
-   **MongoDB Compass** – GUI for managing the database.
-   **Postman** – For testing API endpoints.
-   **Git** – Version control system for source code management.


 ## **Installation & Setup**
1. Make sure install all the required tools and technologies listed above.

2. Clone the repository:
`https://github.com/Arunabha-Chatterjee/ERP-Project.git`

3. Set up the backend:
	```
	cd backend 
	mvn clean install
	```
4. Set up the frontend:
	```
	cd frontend
	npm install
	npm run dev
	```
5.   Access the application at:
`http://localhost:{your port number}`

## **Usage**
-  View business data, key metrics, and recent activities at a glance.
-  Add, edit, and delete products, and track product details.
-  Maintain customer information, view customer history, and update records.
-  Create, view, and track invoices and associated transactions.
-  Securely log-in and registration for users.
-  Receive alerts and confirmations for important actions through toast messages.
-  Navigate the system easily for different screen size's devices.
